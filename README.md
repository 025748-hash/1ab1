Git Learning Project (lab1)

Опис проєкту  
Цей репозиторій створено в рамках практичного вивчення Git та GitHub.  
Тут виконувались лабораторні роботи з контролю версій, гілкування та командної роботи.

Вивчені теми

Модуль 1 — Основи Git  
- Ініціалізація репозиторію  
- Робота з файлами (git status, git add, git commit)  
- Історія комітів (git log, git diff)  
- Ігнорування файлів (.gitignore)

Гілкування та конфлікти  
- Створення та робота з гілками (git branch, git switch)  
- Merge та Fast-forward merge  
- Вирішення merge-конфліктів

Відкат змін  
- git checkout (відновлення файлів)  
- git revert (безпечне скасування комітів)  
- git reset (зміна історії)  
- git commit --amend  

GitHub та віддалені репозиторії  
- Підключення remote (git remote add)  
- git push / git pull / git fetch  
- Робота з GitHub репозиторіями  

Pull Requests та Forks  
- Створення Fork  
- Робота з Pull Request  
- Процес Code Review  

Просунуті техніки  
- git stash (тимчасове збереження змін)  
- git rebase (переписування історії)  
- git cherry-pick (перенесення комітів)
